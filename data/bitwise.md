# 集合&集合

反码~S

补码-S=~S+1

| 术语 | 集合 | 位运算 | 集合示例 | 位运算示例 |
| ---- | ---- | ---- | ---- | ---- |
| 交集 | A ∩ B | a & b | {0,2,3} ∩ {0,1,2} = {0,2} | 1101 & 0111 = 0101 |
| 并集 | A ∪ B | a | b | {0,2,3} ∪ {0,1,2} = {0,1,2,3} | 1101 | 0111 = 1111 |
| 对称差 | A Δ B | a ^ b | {0,2,3} Δ {0,1,2} = {1,3} | 1101 ^ 0111 = 1010 |
| 差 | A \ B | a & ~b | {0,2,3} \ {1,2} = {0,3} | 1101 & 1001 = 1001 |
| 差（子集） | A \ B | a ^ b | {0,2,3} \ {0,2} = {3} | 1101 ^ 0101 = 1000 |
| 包含于 | A ⊆ B | a&b==a | {0,2} ⊆ {0,2,3} | 0101 & 1101 = 0101 |

# 集合&元素

| 术语                 | 集合表示                     | 位运算表示               | 集合示例                                       | 位运算示例                                       |
| -------------------- | ---------------------------- | ------------------------ | --------------------------------------------- | ----------------------------------------------- |
| 空集                 | ∅                            | 0                        |                                                |                                                 |
| 单元素集合           | {i}                          | 1 << i                   | {2}                                            | 1 << 2                                          |
| 全集                 | U = {0,1,2,...,n-1}          | (1 << n) - 1             | {0,1,2,3}                                      | (1 << 4) - 1                                    |
| 补集                 | U\S                          | ((1 << n) - 1) ^ s       | U={0,1,2,3} 补集{1,2}={0,3}                    | 1111 ^ 0110 = 1001                              |
| 属于                 | i ∈ S                        | (s >> i) & 1 == 1        | 2 ∈ {0,2,3}                                    | (1101 >> 2) & 1 = 1                             |
| 不属于               | i ∉ S                        | (s >> i) & 1 == 0        | 1 ∉ {0,2,3}                                    | (1101 >> 1) & 1 = 0                             |
| 添加元素             | S ∪ {i}                      | s | (1 << i)             | {0,3} ∪ {2}                                    | 1001 | (1 << 2)                                    |
| 删除元素             | S \ {i}                      | s & ~(1 << i)            | {0,2,3} \ {2}                                  | 1101 & ~(1 << 2)                                 |
| 删除元素（一定在集合中） | S \ {i}, i ∈ S             | s ^ (1 << i)             | {0,2,3} \ {2}                                  | 1101 ^ (1 << 2)                                  |
| 删除最小元素         |                              | s & (s - 1)              |                                                | 如 s=1101, s-1=1100, s&(s-1)=1100                |

左移i位，相当于乘$2^i$

当s是2的幂次，$s \& (s - 1)$=0，一定是1000...&01111....

| 术语         | Python                          | C++                      |
| ------------ | ------------------------------- | ------------------------ |
| 集合大小     | `s.bit_count()`                 | `__builtin_popcount(s)`  |
| 二进制长度   | `s.bit_length()`                | `__lg(s) + 1`            |
| 集合最大元素 | `s.bit_length() - 1`            | `__lg(s)`                |
| 集合最小元素 | `(s & -s).bit_length() - 1`     | `__builtin_ctz(s)`       |

s & -s保留最右侧1，lowbit操作
